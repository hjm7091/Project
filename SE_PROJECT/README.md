## 소프트웨어공학 팀프로젝트(2019/09 ~ 2019/12)

### 1. 주제
##### 자쥐방 공유 앱(KangwonBnB) : 강원대생을 위한 자취방 이용 서비스

### 2. 개발 목적
##### 장기간 자취방을 비워둬야 하는 상황이나 급하게 일정 기간 방을 빌리고 싶을 때 앱을 이용하여 자취방 거래 가능

### 3. 개발 범위
##### 회원가입, 로그인, 로그아웃, 자취방게시판, 자유게시판
